## Why this repo?

Developers juggle many AI tools. These prompts are **copy-paste ready** examples inspired by real work scenarios — emails, landing pages, SQL, code review, and more.

> **Go to [AI Plaza](https://aiplaza.app) for one-click tools + 190+ models in one account.**

## What's included

- **24** ready-to-use prompt templates (Markdown)
- Machine-readable index: [`prompts.json`](prompts.json)
- Categories: Writing · Marketing · Business · Productivity · Coding

## What's NOT included

- No API keys, tokens, or `.env` files
- No auth, billing, Stripe, Clerk, or database code
- No private SaaS source code or internal routing
- These are **educational community examples**, not a dump of hosted product internals
