# Contributing

Thanks for helping grow a clean, public prompt gallery for AI Plaza.

## Acceptable contributions

- Generic writing / marketing / business / coding / productivity prompts
- Clear system prompt + example user message
- English first (other languages welcome as extras)

## Not acceptable

- API keys, tokens, customer data, internal docs
- Full private product source or billing/auth code
- Legal/medical/financial advice marketed as professional counsel
- Spam or scraped dumps of proprietary hosted workflows

## Format

Add `prompts/<category>/<id>.md` using existing files as a template, then add an entry to `prompts.json`.
